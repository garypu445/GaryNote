Django Broken pipe


Django Broken pipe錯誤

如圖錯誤所示

Broken pipe錯誤，通常都難以猜測其問題，比如運算量太大，耗時太長等。

(1）broken pipe的字面意思是“管道破裂”。broken pip的原因是該管道的讀端被關閉。

(2）broken pipe經常發生socket關閉之後（或者其他的描述符關閉之後）的write操作中。

(3）發生broken pipe錯誤時，進程收到SIGPIPE信號，默認動作是進程終止。

(4）broken pipe最直接的意思是：寫入端出現的時候，另一端卻休息或退出了，因此造成沒有及時取走管道中的數據，從而係統異常退出。
